Visualizaciones

En esta sección puedes consultar Datos Abiertos de forma visual. Aquí encontrarás numerosas visualizaciones de las bases de datos más descargadas. Para generar una visualización, primero elige un tipo de gráfica y luego selecciona la base de datos.


1. Selecciona un tipo de gráfica

  • Gráfica de barras
  • Gráfica de composición
  • Mapa
  • Gráfica de sectores
  • Gráfica de dispersión
2. Selecciona la base de datos

Ilustran longitudes de barras proporcionales a valores relacionados mediante un eje de “X” y un eje de “Y”. Es útil para describir incrementos, disminuciones o estabilidades de valores, variables, indicadores, cantidades, porcentajes, como número de ventas por año, incremento o disminución del PIB, de exportaciones, importaciones, decomisos, etc durante un periodo de tiempo determinado.

Utilizada para describir de forma puntual y detallada las características cualitativas o cuantitativas de un recurso dentro de una categoría determinada. Cada segmento contiene N cantidad de niveles con información específica. Por ejemplo, las características poblacionales, económicas o demográficas de un sector determinado de la población.

Utilizado para describir, relacionar, analizar o correlacionar características o variables cualitativas y cuantitativas asociadas al espacio y territorio respecto a un área geográfica en específico. Por ejemplo cobertura de agua potable respecto al territorio nacional, estatal o municipal.

Se utiliza para describir variables cualitativas o categóricas dentro de un círculo, el cual está dividido en partes, así se puede saber la proporción que guardan las partes del círculo en relación al número de datos de cada categoría. Por ejemplo, en una universidad al representar su población dentro de un pie chart se podrá saber qué proporción del círculo tienen los estudiantes, profesores, trabajadores, intendentes, etc.

Se utiliza para identificar relaciones o supuestos entre dos variables, particularmente útil para la interpretación de datos. Por ejemplo, el aumento o disminución o estabilidad de un precio en una región económica en relación con una línea de tiempo determinada.
