Visualizaciones

En esta sección puedes consultar Datos Abiertos de forma visual. Aquí encontrarás numerosas visualizaciones de las bases de datos más descargadas. Para generar una visualización, primero elige un tipo de gráfica y luego selecciona la base de datos.

1. Selecciona un tipo de gráfica
  • Gráfica de barras
  • Gráfica de composición
  • Mapa
  • Gráfica de sectores
  • Gráfica de dispersión
2. Selecciona la base de datos
